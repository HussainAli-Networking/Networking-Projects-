# Networking Projects

This repository contains hands-on networking projects and lab implementations focused on enterprise network design, WAN technologies, and security using defense-in-depth principles.

All projects are built, configured, tested, and verified in simulated environments, primarily using **EVE-NG**, with an emphasis on real-world behavior, verification commands, and documentation.
